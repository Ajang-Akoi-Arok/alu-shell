#!/bin/bash

This project contains scripts for learning basic Linux shell commands.

## Files Description

0-current_working_directory - Prints the absolute path of current directory
1-listit - Lists contents of current directory
2-bring_me_home - Changes to home directory
3-listfiles - Lists contents in long format
4-listmorefiles - Lists all files including hidden ones
5-listfilesdigitonly - Lists with numeric user/group IDs
6-firstdirectory - Creates directory in /tmp/
7-movethatfile - Moves file to created directory
8-firstdelete - Deletes a file
9-firstdirdeletion - Deletes a directory
10-back - Returns to previous directory
11-lists - Lists multiple directories
12-file_type - Checks file type
13-symbolic_link - Creates symbolic link
14-copy_html - Copies HTML files
15-lets_move - Moves uppercase files
16-clean_emacs - Removes backup files
17-tree - Creates nested directories
